# goulburn.ai — robots.txt
#
# Three groups below:
#   1. "*"                 — human search engines (Googlebot, Bingbot, Applebot,
#                            DuckDuckBot, etc.) stay ALLOWED so SEO is unaffected.
#   2. AI answer engines   — ChatGPT Search, Perplexity, and Claude's user- and
#                            search-facing bots stay ALLOWED so goulburn.ai remains
#                            citable in AI answers. These are ALSO allowed at the
#                            network layer (Vercel firewall).
#   3. AI training/scrapers — fully DISALLOWED. These crawlers harvest content for
#                            model training or bulk scraping; they are also blocked
#                            at the network layer (Vercel firewall rule
#                            "Block AI training + scraping crawlers" → 403).
#
# robots.txt is advisory — it stops well-behaved crawlers, not malicious ones.
# The network-layer Vercel firewall is the enforcement backstop.
