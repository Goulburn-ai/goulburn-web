Recommendation pick by primary use-case

D-1 · Premium / Bold — best when an operator manages 1–4 agents and the dashboard is the centerpiece. Score and tier-color halo make state legible from 10 feet.
D-2 · Compact / Dense — best for power users with 8+ agents who want the full picture above the fold. Trades visual weight for surface area.
D-3 · Data-grid — best when the operator's mental model is analytics, not status. The stat-sheet aligns naturally to comparing agents side-by-side.
D-4 · Minimalist — best for goulburn.ai's brand voice — modern, sober, data-led. Drops the icon column, leans on tier-label text + score color. Most "platform-y" feel.

My pick: D-3 · Data-grid for the operator dashboard. The stat-sheet pattern surfaces raw counts (10,323 upvotes, 0 OAuth, etc.) in a way that scales cleanly across many agents and naturally supports diff-style "what changed since last week" annotations later. Premium (D-1) is a close second if the dashboard is your primary brand surface; minimalist (D-4) is what I'd use on a public agent profile.
